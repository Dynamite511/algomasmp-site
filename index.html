How to Join

Algoma SMP is a cross-play Minecraft server, meaning Java Edition and Bedrock Edition players can play together!

🖥️ Java Edition

  1. Open Minecraft Java Edition.
  2. Click Multiplayer.
  3. Click Add Server.
  4. Go to the Algoma SMP website and copy the Java IP address.
  5. Paste the IP into the Server Address box.
  6. Click Done.
  7. Select Algoma SMP and click Join Server.

📱 Bedrock Edition

  1. Open Minecraft Bedrock Edition.
  2. Click Play.
  3. Go to the Servers tab.
  4. Select Add Server.
  5. Go to the Algoma SMP website and copy the Bedrock IP address and Port.
  6. Paste them into the corresponding fields.
  7. Click Save.
  8. Select Algoma SMP and join the server!

🎮 Play Together!

That's it! You can now play Algoma SMP with your friends, whether they're playing on Java Edition or Bedrock Edition.

Make sure you copy the IP addresses directly from the Algoma SMP website so you always have the correct server information.
